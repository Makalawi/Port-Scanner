# 🌐 Port Scanner

A beginner-friendly Python port scanner that checks whether selected TCP ports on a host are open.

## Features

- Scan a range of ports
- Display open ports
- Adjustable timeout
- Lightweight and fast

## Requirements

- Python 3.x

## How to Run

```bash
python port_scanner.py
```

## Example

```
Enter IP:
192.168.1.1

Scanning...

Port 22 Open
Port 80 Open
Port 443 Open
```

## Skills Learned

- Socket programming
- Loops
- Network communication
- TCP connections
